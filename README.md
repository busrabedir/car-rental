## 🚗 Car Rental App

Modern ve kullanıcı dostu bir araç kiralama uygulamasıdır. Kullanıcılar farklı marka, model ve yıl filtreleriyle araçları keşfedebilir ve detaylı bilgilerine ulaşabilir. Dinamik veri çekimi ve akıcı animasyonlarla zenginleştirilmiş bir deneyim sunar.

## ✨ Özellikler

- 🔍 Marka, model ve yıl bazlı gelişmiş filtreleme
- 📄 Sayfalama (pagination) ile performanslı listeleme
- 🚗 Araç detaylarını modal üzerinden görüntüleme
- ⚡ Dinamik API entegrasyonu ile gerçek zamanlı veri
- 🎨 Modern UI ve akıcı animasyonlar
- 📱 Responsive tasarım


## 🛠️ Kullanılan Teknolojiler

- **⚛️ React + TypeScript** 
- **🎨 Tailwind CSS**  
- **🔄 React Router DOM**  
- **🎞️ Framer Motion (motion)** 
- **🔍 React Select**  
- **📄 React Paginate**  
- **🌐 Axios**
